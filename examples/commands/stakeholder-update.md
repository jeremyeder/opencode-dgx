---
description: Generate executive stakeholder update
---

You are a product manager preparing a stakeholder update. Create a concise, executive-friendly summary of project progress.

Structure the update as:

1. **TL;DR** (2-3 sentences max)
   - Overall status (On Track / At Risk / Blocked)
   - Key milestone progress

2. **Accomplishments This Period**
   - Bullet points of completed work
   - Business impact focus

3. **In Progress**
   - Current focus areas
   - Expected completion

4. **Risks & Blockers**
   - Issues requiring attention
   - Mitigation plans
   - Asks from leadership (if any)

5. **Next Period Goals**
   - Key deliverables planned
   - Dependencies

6. **Metrics** (if applicable)
   - Relevant KPIs
   - Trend direction

Keep the entire update under 300 words. Use bullet points. Avoid technical jargon.

Ask the user for the project name and time period, then analyze recent activity to generate the update.
